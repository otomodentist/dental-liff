# 授業のオトモ — サービス仕様書

## サービス概要

**授業のオトモ**は、歯学部の授業をLINEとAIでリアルタイムに盛り上げる集中度モニタリングサービスです。  
先生が授業中に問題を出題し、学生はLINEで回答するだけ。GPT-4o-miniが即座に採点し、集中度スコアとクラスランキングをその場で表示。授業が競い合いの場になります。

---

## ターゲットユーザー

### 一次ターゲット：歯学部の先生（購入者）
- 歯学部専任講師・教授
- 授業の質と学生の集中度に課題意識がある
- LINEやスマホ操作は基本的にできる

### 二次ターゲット：歯学部の学生（利用者）
- 歯学部 1〜6年生
- LINEは日常的に使っている
- 授業中のスマホ使用に後ろめたさがある → 公認のエンゲージメントに転換

### 三次ターゲット：保護者
- 学生の保護者（任意参加）
- 子どもの授業参加・学習状況をLINEで把握したい

---

## 解決する課題

| 課題 | 従来の状況 | 授業のオトモ |
|---|---|---|
| 学生の集中力低下 | 授業中のスマホ・内職・居眠り | リアルタイム出題で「今聞いてないと答えられない」状況を作る |
| 回答の積極性低下 | 手を挙げて発言するのは恥ずかしい | LINEで手軽に送信、みんなの回答が見える |
| 理解度の把握困難 | 先生が学生の理解度を掴めない | 集中度スコアで授業ごと・学生ごとに可視化 |
| 教員の準備負担 | 毎回の出題内容をその場で入力 | 問題バンクで事前登録、授業中は1タップで出題 |
| 保護者との連携不足 | 授業参加状況が保護者に届かない | コマ終了時に採点サマリーをLINEで自動通知 |

## 主要機能

### 1. リアルタイム出題 & AI採点
- 教員が管理画面または教員ポータルから問題文と模範解答を入力して出題
- 学生はLINEのチャット画面に回答を打ち込むだけ
- GPT-4o-miniが「意味の正確さ」で0〜100点採点
- 複数回答・OR表記・漢字誤字許容・アルファベット大文字小文字不問など柔軟な採点

### 2. 集中度スコア
- **集中度スコア = 正確性スコア × 速度乗数**
- 速度乗数：60秒以内=×1.0、180秒以降=×0.1（線形減少）
- 「授業をちゃんと聞いていれば素早く答えられるはず」という考え方
- 一コマの集中度は幾何平均（維持スコア）で評価 — 1問でも低いと大きく下がる

### 3. LIVEクラスランキング
- 出題中はリアルタイムで順位が更新
- 名前・アイコン・回答テキスト・スコアが全員に見える
- 未回答者も「未回答」で表示され、全員が参加している感を演出
- 授業後は一コマ通じた維持スコアでセッションランキングを表示

### 4. 学生マイページ（LINE LIFF）
- 科目ごと・授業回数ごとの集中度推移をグラフで確認
- 過去の問題と自分の回答・スコアを見返せる
- 保護者機能の設定・管理

### 5. 教員ポータル
- 科目ごとのパスワード認証でアクセス
- **問題バンク**：授業前に問題・模範解答を登録。授業中は選んで1タップで出題
- 授業開始・終了・出題操作
- **出題統計**：問題ごとの平均スコア・高得点率・回答数を一覧で確認
- 小テスト管理（日程・範囲の登録 → 学生へLINE通知）
- 「実習：〇〇」形式の関連科目を同一ログインで管理

### 6. 保護者機能
- 学生がシェアURLを保護者に送信 → LINEで登録（最大3名）
- 通知設定（保護者が選択）：
  - 回答ごとに採点結果を通知
  - コマ終了時にサマリー通知（推奨）
  - 通知なし
- 保護者はLINEから子どもの学習状況・スコアを閲覧可能

### 7. 学年管理者ダッシュボード
- 学生一覧・課金ステータス・参加状況の確認
- 科目・学年・学期の管理
- 登録者への一斉メッセージ配信
- 学年管理者（サブ管理者）への権限付与

### 8. AI試験対策プランナー *(アップデート予定)*
- 試験日・科目を登録すると、授業回数ぶんの復習スケジュールを自動生成
- 毎朝LINEで今日の復習タスクを通知
- 「学習済み」ボタンで進捗管理、リプランも1タップ

## ユーザーフロー

### 授業当日（教員）
```
教員ポータルを開く（科目パスワードでログイン）
→ 問題バンクから出題する問題を選ぶ、または新規入力
→ 「授業開始」→「出題」
→ リアルタイムランキング・回答状況を確認しながら授業進行
→ 「授業終了」（全学生・保護者にLINE通知）
```

### 授業当日（学生）
```
LINEに問題が届く
→ チャットに回答を打ち込む
→ 即座にフィードバック返信（正確性・速度・集中度スコア）
→ マイページを開くとリアルタイムランキングが表示されている
→ 授業後、維持スコアでセッションランキングを確認
```

### 授業当日（保護者）
```
授業終了後、LINEにコマ終了サマリーが届く
  「〇〇科目 第N回が終了しました
   集中度維持スコア: XX/100  クラス順位: N位 / M人中」
→ マイページから詳細を確認（任意）
```

## 利用イメージ（コピー素材）

- 「授業中にLINEを送るだけで、AIが採点して即座にフィードバック」
- 「クラス全員の回答がリアルタイムでランキング表示。授業が盛り上がる」
- 「先生が学生の集中度を授業ごとに可視化。どのタイミングで集中が切れたかわかる」
- 「問題バンクで事前準備。授業中は1タップで出題できる」
- 「保護者にも授業参加状況がLINEで届く」
- 「スマホを使うことが授業参加の証明になる」

---

## 技術構成

- **学生・教員の操作画面**：LINE LIFF（LINEアプリ内で動作するWebアプリ）
- **バックエンド**：Google Apps Script（サーバーレス）
- **AI採点**：OpenAI GPT-4o-mini
- **データ保存**：Google Sheets
- **通知**：LINE Messaging API（Multicast / Push）

---

## 料金・提供形態（想定）

- 学年・学期単位での契約
- 登録学生数・科目数・教員数に上限なし
- 保護者機能は追加料金なし（学生登録に付帯）
