# 授業のオトモ — サービス仕様書（HP制作用）

## サービス概要

**授業のオトモ**は、歯学部の授業をLINEとAIでリアルタイムに盛り上げる集中度モニタリングサービスです。
先生が授業中に問題を出題し、学生はLINEで回答するだけ。GPT-4oが即座に採点し、集中度スコアとクラスランキングをその場で表示。授業が競い合いの場になります。

---

## ターゲットユーザー

### 一次ターゲット：歯学部の先生（購入者）
- 歯学部専任講師・教授
- 授業の質と学生の集中度に課題意識がある
- LINEやスマホ操作は基本的にできる

### 二次ターゲット：歯学部の学生（利用者）
- 歯学部 1〜6年生
- LINEは日常的に使っている
- 授業中のスマホ使用に後ろめたさがある → 公認のエンゲージメントに転換

---

## 解決する課題

| 課題 | 従来の状況 | 授業のオトモ |
|---|---|---|
| 学生の集中力低下 | 授業中のスマホ・内職・居眠り | リアルタイム出題で「今聞いてないと答えられない」状況を作る |
| 回答の積極性低下 | 手を挙げて発言するのは恥ずかしい | LINEで匿名っぽく送信、みんなの回答が見える |
| 理解度の把握困難 | 先生が学生の理解度を掴めない | 集中度スコアで授業ごと・学生ごとに可視化 |
| 試験対策の非効率 | 復習タイミングが学生まかせ | AIが試験日から逆算して復習スケジュールを自動生成 |

---

## 主要機能

### 1. リアルタイム出題 & AI採点
- 先生が管理画面（LIFF）から問題文と模範解答を入力して出題
- 学生はLINEのチャット画面に回答を打ち込むだけ
- GPT-4o-miniが「意味の正確さ」で0〜100点採点（キーワード一致ではなく理解度で判定）
- 逆のことを言っていれば低得点、似た言葉でも意味が違えば加点しない

### 2. 集中度スコア
- **集中度スコア = 正確性スコア × 速度乗数**
- 速度乗数：60秒以内=1.0、以降は遅いほど下がり4.5分以降=0.3
- 「授業をちゃんと聞いていれば素早く答えられるはず」という考え方
- 一コマの集中度は幾何平均（維持スコア）で評価 — 1問でも低いと大きく下がる

### 3. LIVEクラスランキング
- 出題中はリアルタイムで順位が更新（5秒ごと）
- 名前・アイコン・回答テキスト・スコアが全員に見える
- 未回答者も「未回答」で表示され、全員が参加している感を演出
- 授業後は一コマ通じた維持スコアでセッションランキングを表示

### 4. 学生マイページ（LINE LIFF）
- 科目ごと・授業回数ごとの集中度推移をグラフで確認
- 今日・明日の復習スケジュールを表示
- 過去の問題と自分の回答・スコアを見返せる

### 5. AI試験対策プランナー
- 試験日・科目を登録すると、授業回数ぶんの復習スケジュールを自動生成
- 復習→再復習の2段階スケジュール
- 「学習済み」ボタンで進捗管理、リプランも1タップ

### 6. 管理画面（先生用 LIFF）
- 授業開始・出題・終了をボタン操作だけで実施
- 学生一覧・参加状況・スコアをリアルタイム確認
- 試験日程・復習量の登録・編集
- ランキング・回答状況を先生側でも把握

## ユーザーフロー

### 授業当日（先生）
```
管理画面を開く
→ 科目・回数を選んで「授業開始」（全学生にLINE通知）
→ 問題文・模範解答を入力して「出題」（全学生にLINEで送信）
→ リアルタイムランキングを確認しながら授業進行
→ 「授業終了」（全学生にLINE通知）
```

### 授業当日（学生）
```
LINEに問題が届く
→ チャットに回答を打ち込む
→ 即座にフィードバック返信（正確性・速度・集中度スコア）
→ マイページを開くとリアルタイムランキングが表示されている
→ 授業後、維持スコアでセッションランキングを確認
```

### 試験前（学生）
```
マイページ → 試験対策 → 試験日を確認
→ AIが生成した復習スケジュールを確認
→ 毎朝LINEで今日の復習タスクが届く
→ 「学習済み」を記録して進捗管理
```

- 「授業中にLINEを送るだけで、AI が採点して即座にフィードバック」
- 「クラス全員の回答がリアルタイムでランキング表示。授業が盛り上がる」
- 「試験日を入れるだけで、復習スケジュールが自動で出てくる」
- 「先生が学生の集中度を授業ごとに可視化。どのタイミングで集中が切れたかわかる」
- 「スマホを使うことが授業参加の証明になる」
