# 授業のオトモ — サービス仕様書

## サービス概要

**授業のオトモ**は、歯学部の授業をLINEとAIでリアルタイムに活性化する授業運営インフラです。

国家試験合格実績の向上と定員募集力の強化を目指す歯科大学・歯学部に向け、  
「目の前の授業に学生を集中させる」ことを起点に、日々の授業運営から学習管理までを一括サポートします。

---

## 導入対象

### 意思決定者（購買者）
- 歯科大学・歯学部の学長・学部長・教務部門
- 国家試験合格率の改善・定員充足に課題を持つ大学執行部
- 授業の質向上を学校経営上の優先課題と捉えている担当者

### 利用者
- 各科目の担当教員（教員ポータルから問題出題・統計確認）
- 学生（LINEで回答・マイページで自己管理）
- 保護者（任意登録・授業参加状況の通知受信）

---

## 解決する課題

| 学校が抱える課題 | 授業のオトモのアプローチ |
|---|---|
| 国家試験合格率が伸びない | 毎回の授業で理解度をリアルタイム計測・蓄積し、どの科目・どの回で学生が脱落しているかを可視化 |
| 授業中の学生の集中力低下 | LINEによるリアルタイム出題で「今聞いていないと答えられない」状況を作り、主体的な参加を促す |
| 教員の授業改善が属人的 | 出題統計（平均スコア・高得点率・回答数）でどの問題・どのコマが難しいかをデータで把握 |
| 学生の自己管理が難しい | 集中度スコアの推移・過去問の振り返りをLINEマイページで常に確認できる |
| 定員募集・受験者確保 | 高い合格実績と「授業の質」を可視化することでブランド訴求の根拠になる |

## 主要機能

### 1. リアルタイム出題 & AI採点
- 教員が管理画面または教員ポータルから問題文と模範解答を入力して出題
- 学生はLINEのチャット画面に回答を打ち込むだけ
- GPT-4o-miniが「意味の正確さ」で0〜100点採点
- 複数回答・OR表記・漢字誤字許容・アルファベット大文字小文字不問など柔軟な採点

### 2. 集中度スコア
- **集中度スコア = 正確性スコア × 速度乗数**
- 速度乗数：60秒以内=×1.0、180秒以降=×0.1（線形減少）
- 「授業をちゃんと聞いていれば素早く答えられるはず」という考え方
- 一コマの集中度は幾何平均（維持スコア）で評価 — 1問でも低いと大きく下がる

### 3. LIVEクラスランキング
- 出題中はリアルタイムで順位が更新
- 名前・アイコン・回答テキスト・スコアが全員に見える
- 未回答者も「未回答」で表示され、全員が参加している感を演出
- 授業後は一コマ通じた維持スコアでセッションランキングを表示

### 4. 学生マイページ（LINE LIFF）
- 科目ごと・授業回数ごとの集中度推移をグラフで確認
- 過去の問題と自分の回答・スコアを見返せる
- 保護者機能の設定・管理

### 5. 教員ポータル
- 科目ごとのパスワード認証でアクセス
- **問題バンク**：授業前に問題・模範解答を登録。授業中は選んで1タップで出題
- 授業開始・終了・出題操作
- **出題統計**：問題ごとの平均スコア・高得点率・回答数を一覧で確認
- 小テスト管理（日程・範囲の登録 → 学生へLINE通知）
- 「実習：〇〇」形式の関連科目を同一ログインで管理

### 6. 保護者機能
- 学生がシェアURLを保護者に送信 → LINEで登録（最大3名）
- 通知設定：回答ごとに採点結果 / コマ終了時サマリー（推奨）/ なし
- 保護者はLINEから子どもの学習状況・スコアを閲覧可能

### 7. 管理者ダッシュボード

#### 総合管理者（教務部門・学校全体の管理者）
- 全学生の参加状況・スコア一覧（ダッシュボード）
- 学生登録管理・一斉メッセージ配信
- 科目・学年・学期の管理
- サブ管理者（学年主任など）への権限付与

#### サブ管理者（学年主任など）
- 担当学年の授業リアルタイムランキング・回答状況を閲覧（読み取り専用）
- 試験日程・小テストの確認
- 問題配信・学生管理操作は不可

### 8. AI試験対策プランナー *(アップデート予定)*
- 試験日・科目を登録すると、授業回数ぶんの復習スケジュールを自動生成
- 毎朝LINEで今日の復習タスクを通知
- 「学習済み」ボタンで進捗管理、リプランも1タップ

### 9. AIによる自動問題生成 *(アップデート予定)*
- 教員が授業スライド（PDF）を教員ポータルにアップロード
- AIがスライド内容を解析し、一問一答形式の問題を自動生成
- 生成された問題は教員が確認・編集後に問題バンクへ登録
- 毎回の授業準備にかかる問題作成コストを大幅に削減

## ユーザーフロー

### 授業当日（教員）
```
教員ポータルを開く（科目パスワードでログイン）
→ 問題バンクから出題する問題を選ぶ、または新規入力
→ 「授業開始」→「出題」
→ リアルタイムランキング・回答状況を確認しながら授業進行
→ 「授業終了」（全学生・保護者にLINE通知）
```

### 授業当日（学生）
```
LINEに問題が届く
→ チャットに回答を打ち込む
→ 即座にフィードバック返信（正確性・速度・集中度スコア）
→ マイページを開くとリアルタイムランキングが表示されている
→ 授業後、維持スコアでセッションランキングを確認
```

## 技術構成

- **学生・教員の操作画面**：LINE LIFF（LINEアプリ内で動作するWebアプリ）
- **バックエンド**：Google Apps Script（サーバーレス）
- **AI採点**：OpenAI GPT-4o-mini
- **データ保存**：Google Sheets
- **通知**：LINE Messaging API（Multicast / Push）

## 料金・提供形態（想定）

- **学校・学部単位**での契約（個人契約なし）
- 学年・学期単位のサブスクリプション
- 登録学生数・科目数・教員数に上限なし
- 導入支援・初期設定サポートあり
